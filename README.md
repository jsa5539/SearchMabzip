# SearchMabzip

SearchMabzip is an Android application project for searching and managing restaurant information.
The project is designed as a mobile app that helps users find useful restaurant-related information in a simple and accessible way.

## Features

* Restaurant search feature
* Mobile-friendly Android UI
* Java-based Android application structure
* Gradle-based project setup
* Easy project import through Android Studio

## Tech Stack

* Java
* Android Studio
* Gradle Kotlin DSL
* Android SDK

## Project Structure

```text
SearchMabzip/
├── app/
│   └── src/
├── gradle/
├── build.gradle.kts
├── settings.gradle.kts
└── gradlew
```

## Getting Started

### Prerequisites

Before running this project, make sure you have:

* Android Studio installed
* Android SDK configured
* JDK installed
* A physical Android device or emulator

### Installation

1. Clone the repository.

```bash
git clone https://github.com/jsa5539/SearchMabzip.git
```

2. Open the project in Android Studio.

3. Wait for Gradle sync to finish.

4. Run the app on an emulator or Android device.

## Purpose

This project was created to practice Android app development and implement a restaurant search application using Java and Android Studio.
